laravel homestead 環境快速安裝

2020-04-17
backend, laravel

Laravel 推出 Homestead 搭配 Vagrant Box，讓初學者不用為環境問題而煩惱，減少浪費時間在架設 Laravel。
簡單來說 Laravel Homestead = Vagrant + VirtualBox + Laravel 安裝包。

步驟一：事前準備

  • 安裝 Vagrant
  • 安裝 Virtual Box

Windows 開發環境，多安裝 Bash Tool
完成後透過Vagrant下載封裝包

直接抓取官方 Homestead 程式碼

打開 Homestead.yaml
設定開發虛擬目錄及相對應主機名稱
