顯示陣列內容函式

1.print_r()
可以顯示指定的陣列或物件的內容，但是只能指定一個變數，其格式為 print_r(變數);
2.var_dump()
可以顯示指定的陣列或物件的資料型別、大小及內容，而且能指定多個變數，對於程式測試與偵錯時特別有用。
其格式為 var_dump(變數１[,變數2][…, 變數n]);
3.var_export()
可以將指定的陣列或物件內容以PHP的合法程式碼返回，其格式為：var_export(變數)

print_r()的結果:Array ( [0] => 甲 [1] => 乙 [2] => 丙 )
var_dump()的結果：array(3) { [0]=>string(3) “甲” [1]=>string(3) “乙” [2]=>string(3) “丙” }
var_export()的結果：array ( 0 =>‘甲’, 1 =>‘乙’, 2 =>‘丙’, )

  1. 可以加底線
  2. 英文命名與語意化命名
  3. 大小寫不同視為不同變數

變數的命名與編寫風格

  • 駝峰式大小寫（Camel-Case）
  • 小駝峰式命名法（lower camel case）
    例如：$progressBarTitle。
  • 大駝峰式命名法（upper camel case）
    例如：$ProgressBarTitle。
  • 蛇型命名法(Snake case)
    蛇型命名法(Snake case)在各單字中間以底線區隔。例如：$progress_bar_title
